# Телеграм-бот НКО с ИИ для генерации текста, картинок, постов и контент-плана
<br>

Бот позволяет генерировать при поддержке ИИ, который задаёт уточняющие вопросы:
* текст 
* картинки 
* посты
* контент-план
<br>
<br>

Рекомендуемая версия python: **3.14**

Для запуска бота нужно запустить файл: **src/bot/main.py**

Для добавления API ключа бота его необходимо получить в телеграме у @BotFather, затем создать файл ".env" в корневой папке проекта и вставить в файл ключ.
